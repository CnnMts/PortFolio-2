Caisse Outils

Application de gestion de caisse pour un magasin spécialisé dans les jeux vidéo d’occasion, mangas et jeux de société. Développée en Nuxt + Electron avec génération de factures PDF.

Technologies

  • • Nuxt 3
  • • Electron
  • • TailwindCSS
  • • jsPDF

Fonctionnalités

  • • Gestion du stock
  • • Recherche par code-barres
  • • Suivi des ventes
  • • Génération de factures PDF

Rôle

Développement complet : backend (API PHP, MySQL) et frontend (Nuxt, Tailwind). Intégration Electron pour une application locale installable.
